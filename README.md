## Overview

The Weather Forecast Live App is a modern web application designed to help users stay informed about current weather conditions and forecasts. With its intuitive design and live data updates, users can quickly check weather information for their location without any complications.

This application demonstrates best practices in front-end development with semantic HTML structure and responsive CSS styling that works perfectly on all devices - desktop, tablet, and mobile.

## Features

- **Live Weather Updates** - Get real-time weather information updated instantly as conditions change
- **Responsive Design** - Seamlessly adapts to any screen size (desktop, tablet, mobile)
- **Clean User Interface** - Intuitive and easy-to-navigate weather display
- **Location-Based Weather** - View weather data for your current location or search for other locations
- **Weather Forecast** - Check upcoming weather predictions and plan accordingly
- **Fast Performance** - Lightweight application that loads quickly and runs smoothly

## Technologies Used

- **HTML (90.4%)** - Semantic markup for proper structure and accessibility
  - Clean, semantic HTML5 elements for better SEO and accessibility
  - Proper form handling for user input and location searches
  
- **CSS (9.6%)** - Modern styling for responsive and attractive design
  - Flexible grid and flexbox layouts for responsiveness
  - Media queries for mobile-first design approach
  - Smooth animations and transitions for better user experience

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for fetching weather data

### Installation & Usage

1. **Clone the repository:**
```bash
git clone https://github.com/Sindhumadanala/Weather-Forecast-live-App.git
```

2. **Navigate to the project directory:**
```bash
cd Weather-Forecast-live-App
```

3. **Open the application:**
   - Simply open `index.html` in your web browser
   - No server setup or dependencies required
   - The app will load and be ready to use immediately

4. **Search for weather:**
   - Enter a city name in the search bar
   - View current weather conditions
   - Check the forecast for upcoming days

## How It Works

1. **User Input** - Users enter their desired location in the search bar
2. **Data Fetching** - The app retrieves weather data from a weather API
3. **Processing** - Data is processed and formatted for display
4. **Display** - Current weather and forecast information is rendered on the page
5. **Live Updates** - Weather information updates automatically as new data arrives
