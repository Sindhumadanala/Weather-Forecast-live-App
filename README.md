# Weather Forecast Live App

A live weather forecasting application built with HTML and CSS.

## Overview

This project provides real-time weather information with a clean and responsive user interface.

## Features

- Live weather updates
- Responsive design
- Clean and intuitive UI

## Technologies Used

- **HTML** (90.4%)
- **CSS** (9.6%)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Sindhumadanala/Weather-Forecast-live-App.git
```

2. Navigate to the project directory:
```bash
cd Weather-Forecast-live-App
```

3. Open `index.html` in your web browser to view the application.
